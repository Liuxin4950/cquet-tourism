# 任务：用户端前端项目搭建（Week 1）

## 1. 任务背景

毕业设计重庆文旅项目，用户端前端（Vue3 + Vite + Tailwind + Pinia）需在 3 周内完成交付。本任务为 Week 1 的核心：搭建项目框架、配置依赖、验证关键技术（MapLibre GL JS）。

## 2. 业务流程

```
1. 初始化 Vue3 + Vite 项目（cquet-user/）
2. 安装并配置依赖（Tailwind、Pinia、Vue Router、Axios、Swiper、GSAP、MapLibre）
3. 配置 Tailwind + 设计系统变量（颜色、字体）
4. 配置 Vite proxy（开发环境代理到 localhost:8080）
5. 配置 Vite build.rollupOptions.output.manualChunks（代码分割）
6. 搭建目录结构（api/、stores/、views/、components/、router/）
7. 编写 request.ts（Vue 3 兼容 Axios 封装，含 JWT interceptor）
8. 搭建 Pinia stores（auth、scenicSpot、venue、activity）
9. 编写 Vue Router 配置（含 beforeEach 路由守卫）
10. 验证 MapLibre GL JS 可行性（spike）
```

### API 基础路径
- 开发环境：`/tourism/**` → `http://localhost:8080/tourism/**`（Vite proxy）
- 生产环境：`VITE_API_BASE_URL` 环境变量

### JWT Token 处理
- 存储：`localStorage.getItem('token')`
- 注入：`request.ts` interceptor 在每个请求 header 注入 `Authorization: Bearer ${token}`
- 过期处理：401 响应 → 清除 token → 跳转 `/login`

### MapLibre 配置
- 底图样式：MapTiler 免费瓦片（`https://api.maptiler.com/maps/streets/style.json?key=YOUR_KEY`）或 OpenStreetMap
- 备选：预设重庆区域 bounds `[[106.45, 29.45], [106.65, 29.60]]`
- 标志性建筑坐标（WGS-84）：
  - 洪崖洞：29.5628, 106.5833
  - 解放碑：29.5589, 106.5781
  - 重庆大剧院：29.5650, 106.5764

## 5. 风险点

| 风险 | 影响 | 应对 |
|------|------|------|
| MapLibre 3D 建筑渲染性能差 | 首页效果打折 | 降级为 2D 地图 + 简单 marker |
| MapTiler 需要 API Key | 需用户申请 | 使用 OSM 瓦片作为兜底 |
| Swiper + Tailwind 冲突 | 样式异常 | Week 1 spike 验证 |
| CORS 配置问题 | API 请求失败 | 开发环境用 Vite proxy 绕过 |
| 后端接口未提供游客访问权限 | 前端无法访问数据 | 确认 `@PreAuthorize` 是否需要登录 |

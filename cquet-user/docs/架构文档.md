# 架构文档

> 本文档是项目的技术基线，描述「怎么做」。需求背景请见 `docs/需求文档.md`。

---

## 1. 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 框架 | Vue 3 + Vite | 组合式 API（Composition API）|
| 语言 | TypeScript | 类型安全 |
| 样式 | Tailwind CSS | 原子化 CSS |
| 动画 | GSAP（简单滚动）+ Swiper | 精选推荐轮播 |
| 状态 | Pinia | 比 Vuex 轻量 |
| 路由 | Vue Router | `beforeEach` 路由守卫 |
| HTTP | Axios（自封装 `request.ts`）| 不复用若依的 request.js |
| 图片懒加载 | Native `<img loading="lazy">` | 零依赖 |

> **为什么不复用若依的 request.js？** 若依的 request.js 深度耦合 Vue 2 和其自身的响应式结构，在 Vue 3 环境下不兼容。

## 6. JWT Token 处理

- 若依将 token 放在 **cookie** 或 **header** `Authorization: Bearer {token}`
- Axios 拦截器统一注入：`config.headers['Authorization'] = 'Bearer ' + getToken()`
- Vite dev：proxy 配置 `/tourism/**` → `localhost:8080`
- token 过期时跳转登录页（`beforeEach` 守卫判断 `code === 401`）

## 7. GSAP 简单滚动动画（必须实现）

1. **Hero 标题进入**：`opacity:0, y:30` → `opacity:1, y:0`，duration 0.8s，ease: power2.out
2. **精选推荐卡片**：`opacity:0, y:40` → `opacity:1, y:0`，duration 0.6s，stagger 0.1s
3. **关于重庆数字**：countUp 效果，duration 1.5s
4. **三栏卡片**：hover 时 `scale:1.02, y:-4`，transition 0.25s ease

> **Week 1 Spike**：Day 1 必须验证 Tailwind CSS transitions + GSAP ScrollTrigger + Swiper 三者共存无冲突。若有冲突，优先保 Swiper，GSAP 降级为纯 CSS 动画。

## 8. 开发原则

1. **功能完整优先于功能数量** — 先把每个页面的所有状态（loading/empty/error/success）处理完，再做下一个功能
2. **数据先通，UI 后美** — 先保证 API 数据正确展示，再打磨视觉效果
3. **每个组件做好再下一个** — 不留半成品组件

## 9. 开发顺序建议

**Phase 1：基础设施**
1. 检查现有项目结构，补充缺失的基础文件
2. 配置 Tailwind + Manrope 字体
3. 封装 `request.ts`（Axios + 拦截器 + 401 处理）
4. 实现 Pinia auth store
5. 实现路由守卫（beforeEach）

**Phase 2：核心页面**
6. 首页 HomeView（骨架屏 + mock 数据）
7. 景区 SpotListView + SpotDetailView
8. 场馆 VenueListView + VenueDetailView
9. 活动 ActivityListView + ActivityDetailView

**Phase 3：内容 + 认证**
10. 新闻 NewsListView + NewsDetailView
11. 公告 NoticeListView + NoticeDetailView
12. 登录 LoginView + 注册 RegisterView

**Phase 4：用户中心**
13. ProfileView 框架（Tab 切换）
14. 我的报名（已有接口）
15. 我的收藏（依赖后端新接口）
16. 我的评论（依赖后端新接口）

**Phase 5：打磨**
17. GSAP 简单滚动动画
18. 骨架屏 + loading 状态
19. 空状态 + 错误处理 + Toast 系统
20. 移动端适配细节

## 10. 性能优化

| # | 问题 | 解决方案 |
|---|------|----------|
| P1 | 图片无懒加载 | `<img loading="lazy">` |
| P2 | Bundle 无代码分割 | `manualChunks` in vite.config.ts |
| P3 | Google Fonts 阻塞渲染 | `font-display: swap` + `preconnect` |
| P4 | 加载时白屏 | `<LoadingState>` 骨架屏组件 |

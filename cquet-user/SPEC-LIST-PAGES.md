# 列表页 Hero + 分隔线列表 重设计规格

## 1. 概述与目标

将 SpotListView、VenueListView、ActivityListView 三个列表页改为参考 Range Developments 网站风格：
- Full-viewport Hero Banner（100vh）
- Tab 展示栏（纯装饰）+ 结果计数
- 卡片布局：桌面端 image-left（图片左 | 内容右），移动端垂直堆叠
- 每行卡片之间有分隔线（horizontal divider）

## 2. 视觉语言

### 品牌色（沿用 tailwind.config.js）
```
brand:      #222a2c   (charcoal)
dark:       #092c38   (dark teal)
light:      #f5f4f2   (warm white)
muted:      #648992   (steel grey)
accent:     #105670   (deep teal)
```

## 3. Hero Banner 规格

### 3.1 尺寸与布局
- **高度**: `100vh`（桌面 + 移动端一致）
- **宽度**: `100vw`
- **定位**: `absolute`，`top: 0`，`left: 0`，`z-index: 1`
- **父容器**: `position: relative`，NavBar 在 `z-index: 50`
- **背景**: `object-cover`，聚焦中心
- **叠加层**: 底部渐变遮罩（内联 style）:
  `linear-gradient(to top, rgba(9,44,56,0.88) 0%, rgba(9,44,56,0.5) 40%, transparent 70%)`

### 3.2 文字排版
- **H1 标题**: `font-heading`, 3.5rem（桌面）/ 2rem（移动端），weight 700，白色，左下角
- **副标题**: `font-montserrat`, 0.875rem，白色/80%，字间距 0.15em，左下角，标题下方 12px
- **标题区域内边距**: `pl-8 lg:pl-16`，`pb-12 lg:pb-20`

| SpotListView | `photo-1609609697762-87d2c2c944b4` | A级景区 | 探索重庆，感受山城魅力 |
| VenueListView | `photo-1514539079130-25950c84af65` | 文化场馆 | 发现身边的文体空间 |
| ActivityListView | `photo-1545569341-9eb8b30979d9` | 特色活动 | 体验重庆特色文化活动 |

## 4. Tab 展示栏规格

- **背景**: `bg-dark`（#092c38，深青色）
- **内边距**: `px-6 py-3`
- **Tab 项目**: 水平排列，白色文字，`font-montserrat text-sm`，active tab 有底部白色下划线（2px）
- **纯展示性，无功能**: Tab 项 `<span>`，不可点击
- **移动端**: `overflow-x: auto`，可横向滚动

## 5. Section Header 规格（结果计数栏）

- **上边框**: `border-t border-dark/10`（1px 淡分隔线）
- **背景**: `bg-light`（#f5f4f2）
- **内边距**: `px-6 py-4`
- **左侧**: "{n} 个结果" — `font-montserrat text-sm text-muted`

## 6. 卡片列表规格

### 6.1 桌面端（≥768px）：image-left 水平布局
```
┌─────────────────────────────────────────────────────────┐
│ [IMG 33%]  标题                                         │
│            副标题 / 描述 / 底部信息                      │
└─────────────────────────────────────────────────────────┘
```
- **容器**: `flex flex-row`，`border-t border-dark/10`
- **图片**: `w-1/3`，`aspect-[4/3]`，`object-cover`，`overflow-hidden`
- **图片 hover**: `scale(1.05)`，300ms ease-out
- **内容区**: `w-2/3`，`p-6`，`flex flex-col justify-center`

### 6.2 移动端（<768px）：垂直堆叠
```
┌─────────────────────────────────────────────────────────┐
│ [IMG ——— 全宽，aspect-[16/9] ———]                      │
├─────────────────────────────────────────────────────────┤
│  标题 / 副标题 / 描述                                    │
└─────────────────────────────────────────────────────────┘
```
- **图片**: `w-full aspect-[16/9] object-cover`
- **内容区**: `p-4`

### 6.3 分隔线
- **样式**: `border-t border-dark/10`
- **位置**: 每行卡片之间（border 加在卡片容器上）
- **最后一行**: 无底部边框

## 7. 状态规格

| 状态 | 设计 |
|------|------|
| Loading | Hero + Tab 栏显示 + 3 条 skeleton 卡片（带分隔线） |
| Empty | Hero + Tab + 空状态文案 |
| Error | Hero + Tab + 错误提示 + 重试按钮 |

## 8. 响应式断点

| 断点 | Hero 高度 | 卡片布局 | Tab 栏 |
|------|-----------|----------|--------|
| sm (<640px) | 100vh | 垂直堆叠，全宽 | 可横向滚动 |
| md+ (≥640px) | 100vh | 水平 image-left (1/3 vs 2/3) | 完整显示 |

## 9. 无障碍规格

- Hero 背景图: `aria-hidden="true"`
- Hero 标题: `<h1>`，`font-heading`，≥2rem
- Tab 栏: `<nav aria-label="分类">`
- 卡片: `<article>`，整卡可点击，hover: `outline: 2px solid var(--brand)`

## 10. 实施步骤

1. 更新 `HeroBanner.vue`: `100vh` 高度，`top: 0`，`z-index: 1`，更新渐变色值
2. 创建 `SpotCardRow.vue`: image-left 水平布局 + 分隔线，接收 `spot` prop
3. 同理创建 `VenueCardRow.vue`、`ActivityCardRow.vue`
4. 更新 `SpotListView.vue`: HeroBanner + TabBar（展示性） + SectionHeader + SpotCardRow 列表
5. 同理更新 VenueListView、ActivityListView

## 11. 已确认决策

- ✅ Hero 高度：100vh（全屏）
- ✅ Hero 定位：`top: 0`，`z-index: 1`，在 NavBar 下方
- ✅ Tab 栏：纯展示性，不可点击
- ✅ 分隔线：`border-t border-dark/10`
- ✅ 桌面端卡片：`w-1/3` 图片 + `w-2/3` 内容
- ✅ 移动端卡片：垂直堆叠（全宽）

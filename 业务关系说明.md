# 文旅网站业务关系说明

## 📊 核心业务流程

### 1. 用户浏览流程
```
用户登录 → 浏览景区/场馆 → 查看活动 → 收藏/报名 → 参与互动
```

### 2. 管理员管理流程
```
发布景区信息 → 添加场馆 → 创建活动 → 审核申报 → 管理留言
```

## 🏛️ 业务模块详解

### A级景区 (tourism_scenic_spot)
**定义**：国家旅游局评定的旅游景区等级
- **A级**：基础景区
- **AA级**：良好景区  
- **AAA级**：优秀景区
- **AAAA级**：精品景区
- **AAAAA级**：顶级景区

**特点**：
- 🌄 自然风光或历史文化遗址
- 🎫 可能收费，也可能免费开放
- 📍 地理位置固定，范围较大
- 🚶 游客自由参观

**示例**：
- 重庆洪崖洞（4A级）- 免费开放
- 重庆大足石刻（5A级）- 收费景区
- 武隆天生三桥（5A级）- 收费景区

### 场馆 (tourism_venue)
**定义**：提供特定服务的文化、体育、教育设施

**类型**：
- 🏛️ **博物馆**：重庆中国三峡博物馆
- 📚 **图书馆**：重庆图书馆
- 🏟️ **体育馆**：重庆奥体中心
- 🎭 **剧院**：重庆大剧院
- 🎨 **文化馆**：各区县文化馆

**特点**：
- 🏢 室内场所，有容量限制
- 📅 特定开放时间
- 🎟️ 可能需要预约或购票
- 🎯 提供专业服务

### 特色活动 (tourism_activity)
**定义**：在景区或场馆举办的文化、娱乐、教育活动

**不是卖票业务，而是文化体验！**

#### 活动类型示例：

**文化活动**：
- 🎭 川剧变脸表演（洪崖洞）
- 📜 书法体验课（文化馆）
- 🏺 陶艺制作（博物馆）

**体育活动**：
- 🏃 重庆马拉松（城市道路）
- 🏊 游泳比赛（体育馆）
- 🧘 太极拳晨练（公园）

**节庆活动**：
- 🎊 春节庙会（景区）
- 🥮 中秋赏月（观景台）
- 🐉 端午龙舟赛（江边）

**教育活动**：
- 👨‍👩‍👧‍👦 亲子科普（博物馆）
- 📖 历史讲座（图书馆）
- 🎨 美术展览（美术馆）

## 🔗 表关系说明

### 核心关联关系
```
sys_user (若依用户表)
    ↓
tourism_user_collection (用户收藏)
    ↓
tourism_scenic_spot (景区)
tourism_venue (场馆)
tourism_activity (活动)
    ↓
tourism_activity_application (活动申报)
tourism_activity_comment (活动留言)
```

### 详细关系说明

1. **场馆 → 活动**
   ```sql
   tourism_activity.venue_id → tourism_venue.id
   ```
   - 活动在特定场馆举办
   - 一个场馆可以举办多个活动

2. **用户 → 收藏**
   ```sql
   tourism_user_collection.user_id → sys_user.user_id
   tourism_user_collection.target_id → 景区/场馆/活动的ID
   ```
   - 用户可以收藏景区、场馆、活动
   - 通过target_type区分收藏类型

3. **用户 → 活动申报**
   ```sql
   tourism_activity_application.applicant_id → sys_user.user_id
   tourism_activity_application.activity_id → tourism_activity.id
   ```
   - 用户报名参加活动
   - 需要管理员审核

4. **用户 → 活动留言**
   ```sql
   tourism_activity_comment.user_id → sys_user.user_id
   tourism_activity_comment.activity_id → tourism_activity.id
   ```
   - 用户对活动进行评论
   - 支持回复功能（parent_id）

## 💰 收费模式说明

### 景区收费
- **免费景区**：如解放碑步行街、洪崖洞
- **收费景区**：如大足石刻、武隆天坑
- **部分收费**：景区免费，内部项目收费

### 场馆收费
- **免费场馆**：如图书馆、部分博物馆
- **收费场馆**：如某些展览馆、体育馆
- **会员制**：如健身房、游泳馆

### 活动收费
- **免费活动**：如公益讲座、免费表演
- **收费活动**：如培训课程、特殊体验
- **材料费**：如手工制作、烹饪体验

## 🎯 网站价值

这不是一个纯粹的电商网站，而是：

1. **文化传播平台**：展示地方文化特色
2. **公共服务平台**：提供文旅信息服务
3. **互动体验平台**：促进市民文化参与
4. **资源整合平台**：统一管理文旅资源

## 📱 用户使用场景

### 普通市民
- 周末想找地方玩 → 浏览景区场馆
- 看到有趣活动 → 收藏或报名参加
- 参加后有感想 → 留言分享体验

### 游客
- 来重庆旅游 → 查看A级景区
- 了解本地文化 → 参加特色活动
- 需要文化场馆 → 查看开放时间和位置

### 文化工作者
- 组织活动 → 申请场馆和发布活动
- 推广文化 → 通过平台宣传
- 收集反馈 → 查看用户留言和参与情况
